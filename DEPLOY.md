# Установка Med System на компьютеры

## Главное

Python нужен **только на компьютере, где собирается программа**.
На компьютерах медпункта Python не требуется — туда копируется готовая папка.

## Сборка (один раз, на своём компьютере)

1. Запустить `build.bat` двойным кликом.
2. Дождаться надписи «Готово». Результат — папка `dist\MedSystem\`.

Скрипт сам находит Python, ставит зависимости и собирает exe.
Если Python не установлен, скрипт скажет об этом и даст ссылку.

## Установка на рабочий компьютер

1. Скопировать **всю папку** `dist\MedSystem` на целевой ПК
   (например, в `C:\MedSystem`). Можно через флешку или сеть.
2. Запустить `MedSystem.exe`. Для удобства — правой кнопкой →
   «Отправить» → «Ярлык на рабочий стол».
3. При первом запуске база `med_system.db` создастся автоматически
   рядом с exe.

## Обновление версии

Собрать заново, скопировать новую папку поверх старой,
**кроме файла `med_system.db`** — в нём данные. Проще всего:
скопировать `med_system.db` куда-нибудь, заменить папку, вернуть файл.

## Частые проблемы

**«pyinstaller не является командой»** — уже не актуально: новый
`build.bat` вызывает сборщик через `python -m PyInstaller`, это не
зависит от PATH.

**Windows SmartScreen ругается при первом запуске** — нормально для
неподписанных exe. Нажать «Подробнее» → «Выполнить в любом случае».

**Антивирус удаляет exe** — ложное срабатывание, типичное для
PyInstaller. Добавить папку в исключения антивируса.
